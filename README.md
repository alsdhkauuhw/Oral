# Oral
口腔医疗管理系统
## 环境搭建
python: `3.6.0`(由于本项目使用了较为古老的node-nass, 他需要python环境, 他已不再更新)  
java: `1.8`  
vue: `2.6.10`  
## 技术栈
前端：`Vue`  
后端：`Springboot`  
数据库：`Mysql`
## tips!!
我原先将前端与后端全都部署到spring-boot内置的Tomcat服务器上  
如果你希望继续这个风格，你需要使用`npm`的打包命令
    npm run build
然后更改生成的`dist`文件的地址  
## 前端项目地址
就在这个库的`src\main\resources\admin\admin`地址下
